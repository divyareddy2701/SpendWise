📋 CSV Format Guide

Your CSV must have these exact column headers (first row):

title,amount,category,date,recurring
📝title — Expense description (text)
💰amount — Number only, no ₹ symbol (e.g. 450.00)
🏷️category — Food, Transport, Shopping, Health, Utilities, Education, Entertainment, Other
📅date — Format: YYYY-MM-DD (e.g. 2025-03-15)
🔁recurring — true or false (lowercase)
